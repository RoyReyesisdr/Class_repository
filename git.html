Comandos de git

  • git checkout -b nombre_rama: Sirve para crear una nueva rama y cambiarse a la nueva rama.
  • git merge rama_a_juntar: Sirve para juntar la rama rama_a_juntar con la rama en la que te encuentras.
  • git pull: Sirve para actualizar la rama actual con respecto al repositorio remoto.
